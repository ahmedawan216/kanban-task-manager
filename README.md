# Kanban Task Manager 🚀

A streamlined, functional task management dashboard built with **Vanilla JavaScript**. This project focuses on efficient workflow visualization using native browser APIs.

## 🚀 Live Demo
Check out the live application here: https://ahmedawan216.github.io/kanban-task-manager/

## 🌟 Features
* Interactive Drag & Drop: Move tasks seamlessly between "To Do," "In Progress," and "Done" columns.
* Persistent Data: Integrated with **LocalStorage** to ensure your tasks are saved even after refreshing the browser.
* Dynamic Task Management: Easily add, edit, and delete tasks with real-time UI updates.
* Responsive Design: Optimized for various screen sizes using CSS Grid and Flexbox.

## 🛠️ Tech Stack
* HTML5: Semantic structure for accessibility.
* CSS3: Custom styling with a focus on modern, clean layouts.
* JavaScript (ES6+): Logic for drag-and-drop API, event handling, and data persistence.
